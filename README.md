# Gallagher & Mohan - Process Map Solutions Video

Premium B2B Remotion prototype for a 75-90 second product-launch style video.

## Run

```bash
npm install
npm run dev
```

## Render MP4

```bash
npm run render
```

The MP4 renders to `out/process-map-solutions.mp4`.

## Specs

- 1920x1080
- 30 fps
- 84 seconds
- React + TypeScript
- Remotion + Motion for React
